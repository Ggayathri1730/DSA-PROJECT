Intelligent Metro Flow Optimization Platform

This project is a Java-based application developed to demonstrate the practical use of Data Structures and Algorithms (DSA) in managing passenger travel information in a metro system.

The system allows users to input passenger details such as Passenger ID, Name, Start Station, Destination, and Travel Time through a console interface.

The application processes this data using different data structures and algorithms to efficiently store, search, sort, and manage passenger information.

Data Structures Used

Arrays

Linked Lists

Queue

Hash Table

Algorithms Used

Linear Search

Bubble Sort

Features

Add passenger details

Display passenger list

Sort passengers based on travel time

Search passenger using ID

Simulate passenger boarding queue

Fast passenger lookup using hash table

Technologies Used

Java

Data Structures and Algorithms (DSA)
